수정 : 최경수, 김용정(game)

DoDoABC HTML 버전

/layout.html : masterpage 복사본, 사용하지 않음
/include : 리소스, css, javascript 등의 파일 포함
/include/js/common.js : 공통 함수, DB 저장 및 읽기 (todo 채울것)

* 주의사항 : 모든 경로는 상대 경로이나 sightwords1, sightwords2의 listenandrepeat 학습의 DOMAINS(셀바스 라이브러리) 경로는 절대경로로 해야 인식함

* localtest 방법 : /localtest.html 실행하여 값 입력 후 'GO' 버튼 누르면 url 생성 후 /Default.html 호출, Default.html은 세션 저장 후 학습 진행
